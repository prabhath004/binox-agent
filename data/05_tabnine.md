# Tabnine — Privacy-First AI Code Assistant

## Overview
Tabnine is an AI code completion tool founded in 2018 (originally Codota). It focuses on privacy and security, offering models that can run entirely on-premise or on the developer's machine. Tabnine serves over 1 million developers and targets enterprises with strict data policies.

## Product
- AI code completion for 30+ IDEs
- Whole-line and full-function completions
- Chat interface for code explanations and generation
- AI-powered code review
- Protected model: trained only on permissively licensed code
- On-premise deployment option

## Pricing
- Free: basic AI completions
- Pro: $12/month per user
- Enterprise: custom — on-prem, SSO, admin controls, private models

## Differentiation
- Privacy-first: code never leaves your machine (local model option)
- Trained only on permissive open-source licenses
- On-premise deployment for regulated industries
- SOC 2 Type II certified
- No copyright risk from training data

## Risks
- Local models significantly less capable than cloud GPT-4/Claude
- Falling behind in AI capability race
- Small engineering team compared to GitHub/Microsoft
- Limited context window compared to competitors
- Market positioning squeezed between free tools and premium Copilot
