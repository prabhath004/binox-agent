# Replit — AI-Native Cloud IDE

## Overview
Replit is a browser-based IDE and cloud computing platform founded in 2016. It has pivoted heavily toward AI-assisted development with Replit Agent and Ghostwriter. The platform combines coding, hosting, and deployment in a single browser tab. Over 30 million developers have used the platform.

## Product
- Browser-based IDE with 50+ language support
- Replit Agent: autonomous app builder from natural language
- Ghostwriter: AI code completion and chat
- Instant deployment and hosting (replit.dev domains)
- Multiplayer: real-time collaborative editing
- Built-in database, secrets management, package management

## Pricing
- Free: limited compute, community features
- Replit Core: $25/month — advanced AI, boosted compute, always-on repls
- Teams: $40/user/month — organization management, private repls

## Differentiation
- Zero-setup development environment (nothing to install)
- End-to-end: code → build → deploy in one platform
- Replit Agent creates full applications from prompts
- Strong education and beginner market position
- Model marketplace (using multiple AI providers)

## Risks
- Unit economics of free compute tier
- Competing with local IDEs for professional developers
- Replit Agent quality inconsistency for complex applications
- Infrastructure costs from always-on hosting
- Enterprise adoption limited by browser-only model
