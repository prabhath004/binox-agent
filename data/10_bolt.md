# Bolt.new by StackBlitz — AI Full-Stack App Builder

## Overview
Bolt.new is an AI-powered full-stack application builder by StackBlitz, launched in 2024. It uses WebContainers technology to run a complete Node.js development environment in the browser. Users describe an app in natural language and Bolt.new generates, runs, and deploys it — all without leaving the browser.

## Product
- Full-stack app generation from natural language prompts
- In-browser Node.js environment (WebContainers)
- Live preview as code is generated
- Install npm packages, run servers, interact with APIs
- One-click deploy to Netlify
- Supports React, Next.js, Vue, Svelte, and more

## Pricing
- Free: 5 generations/day
- Pro: $20/month — 1,500 tokens/month, increased limits
- Team: $30/user/month

## Differentiation
- WebContainers: full Node.js in the browser (no cloud VM needed)
- Fastest path from idea to deployed app
- No account required to start (instant access)
- Handles both frontend and backend in one flow
- Strong template library for common app types

## Risks
- WebContainers limited to Node.js ecosystem
- Complex apps often need manual debugging after generation
- Token-based pricing can be expensive for iterative development
- Competing with Replit, v0, and Lovable for the same market
- Browser-based limitation for large-scale applications
