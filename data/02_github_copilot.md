# GitHub Copilot — AI Pair Programmer

## Overview
GitHub Copilot is an AI coding assistant developed by GitHub (Microsoft). Launched in 2021, it uses OpenAI Codex models to provide code suggestions inside popular IDEs. It has the largest installed base of any AI coding tool with over 1.8 million paid subscribers as of 2024.

## Product
- Inline code suggestions in VS Code, JetBrains, Neovim, Visual Studio
- Copilot Chat: conversational coding assistance
- Copilot for CLI: natural language to shell commands
- Copilot Workspace: agentic task planning and execution (preview)
- Code review suggestions in GitHub pull requests

## Pricing
- Individual: $10/month or $100/year
- Business: $19/user/month — organization policy management, IP indemnity
- Enterprise: $39/user/month — fine-tuning, knowledge bases, advanced security

## Differentiation
- Massive distribution via GitHub's 100M+ developer base
- Tight integration with GitHub ecosystem (PRs, Issues, Actions)
- IP indemnity for Business/Enterprise customers
- Access to OpenAI's latest models with Microsoft partnership
- Network effects from training on GitHub's public code corpus

## Risks
- Copyright litigation around training data
- Revenue sharing pressure with OpenAI
- Open-source alternatives gaining ground (Cody, Continue)
- Plugin model means limited control over IDE experience
- Privacy concerns with code being sent to cloud
