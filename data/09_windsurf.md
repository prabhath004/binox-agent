# Windsurf (Codeium) — AI-First Code Editor

## Overview
Windsurf is an AI-first code editor built by Codeium, launched in late 2024. Like Cursor, it's a VS Code fork with deep AI integration. Codeium previously offered a free AI autocomplete extension and pivoted to a full editor experience with agentic capabilities.

## Product
- Cascade: agentic coding flow that chains multiple AI actions
- AI autocomplete with Codeium's proprietary models
- Inline editing and chat
- Flows: multi-step automated coding workflows
- VS Code extension compatibility
- Terminal command generation

## Pricing
- Free: generous free tier with AI completions
- Pro: $15/month — premium model access, more Cascade credits
- Enterprise: custom — SSO, admin controls, private deployment

## Differentiation
- Competitive free tier (more generous than Cursor/Copilot)
- Cascade agentic workflows for complex tasks
- Own proprietary models (not fully dependent on OpenAI/Anthropic)
- Lower price point than Cursor Pro
- Fast autocomplete latency

## Risks
- Late entrant into the AI editor market (Cursor has first-mover advantage)
- Brand confusion between Codeium (extension) and Windsurf (editor)
- Proprietary models may lag behind GPT-4/Claude in capability
- Fundraising pressure after large raise at high valuation
- Feature parity gap with Cursor's more mature agent mode
