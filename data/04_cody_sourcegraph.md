# Cody by Sourcegraph — Context-Aware AI Coding Assistant

## Overview
Cody is an AI coding assistant built by Sourcegraph, a code intelligence company founded in 2013. Cody's key differentiator is deep codebase context: it uses Sourcegraph's code graph to understand your entire repository, dependencies, and code relationships before generating suggestions.

## Product
- IDE extensions for VS Code and JetBrains
- Autocomplete with codebase-aware context
- Chat with @-mentions for files, symbols, repositories
- Commands: explain code, generate unit tests, find code smells
- Context fetching from entire codebase (not just open files)
- Works with multiple LLM providers (Claude, GPT-4, Gemini, Mixtral)

## Pricing
- Free: 500 autocomplete/day, 20 chat messages/day
- Pro: $9/month — unlimited autocomplete and chat
- Enterprise: custom pricing — RBAC, SSO, private deployment, custom models

## Differentiation
- Deepest codebase context via Sourcegraph's code graph
- Repository-scale understanding (not limited to open files)
- LLM-agnostic: supports multiple model providers
- Enterprise code search and intelligence platform as foundation
- Strong on large monorepo codebases

## Risks
- Smaller brand awareness compared to Copilot
- Sourcegraph's enterprise focus may limit individual adoption
- Code graph indexing adds setup complexity
- Free tier limitations reduce viral growth
- Competing against deep-pocketed Microsoft/GitHub
