# v0 by Vercel — AI UI Generator

## Overview
v0 is an AI-powered UI generation tool by Vercel, launched in 2023. It generates React components and full page layouts from natural language prompts or image inputs. It targets frontend developers and designers who want to accelerate UI development.

## Product
- Text-to-UI: describe a component, get React/Tailwind code
- Image-to-code: upload a design, get matching implementation
- Iterative editing via chat
- One-click deploy to Vercel
- Supports Next.js, shadcn/ui, Tailwind CSS
- Component library with community-shared generations

## Pricing
- Free: 200 generations/month (with watermark)
- Premium: $20/month — 5,000 generations, no watermark, priority access

## Differentiation
- Best-in-class for UI/frontend code generation
- Tight integration with Vercel/Next.js ecosystem
- Image-to-code capability
- High-quality output using shadcn/ui component library
- Deploy-ready output (not just code snippets)

## Risks
- Narrow focus (frontend/UI only) limits total addressable market
- Dependent on React/Next.js ecosystem
- Output quality varies for complex, interactive UIs
- Competing with general-purpose AI coders (Cursor, Copilot) adding UI features
- Free tier may cannibalize paid conversions
