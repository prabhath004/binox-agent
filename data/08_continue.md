# Continue — Open-Source AI Code Assistant

## Overview
Continue is an open-source AI code assistant that runs as a VS Code and JetBrains extension. Founded in 2023, it lets developers connect any LLM (local or cloud) and customize their AI coding experience. It's the leading open-source alternative to GitHub Copilot.

## Product
- Tab autocomplete with any model
- Chat with codebase context (@-mentions)
- Inline editing via natural language
- Configurable model providers (OpenAI, Anthropic, Ollama, LM Studio, etc.)
- Custom slash commands and context providers
- Local model support via Ollama

## Pricing
- Free and open-source (Apache 2.0 license)
- Continue for Teams: paid tier with admin controls, analytics (pricing TBD)

## Differentiation
- Fully open-source — no vendor lock-in
- Model-agnostic: use any LLM provider or local model
- Highly customizable (config.json for everything)
- Privacy by default when using local models
- Community-driven development with rapid iteration

## Risks
- No built-in model = higher setup friction
- Quality depends on which model user connects
- Monetization unclear for long-term sustainability
- Competing against well-funded proprietary alternatives
- Small team with limited enterprise sales capacity
